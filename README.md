# jban.dev — Personal Portfolio

Personal portfolio website for Jaeyoung (James) Ban, Senior Full Stack Developer based in Auckland, NZ.

Live site: [www.jamesban.com](https://www.jamesban.com)

## Overview

A VS Code-inspired single-page portfolio with a file explorer sidebar, tabbed editor panels, and an interactive terminal. Built with React + Vite + Tailwind CSS.

## Features

- **Interactive terminal** — type commands like `about`, `skills`, `experience`, `achievements`, `contact` with character-by-character streaming output
- **VS Code layout** — sidebar file explorer, tabbed editor, resizable terminal panel
- **Responsive** — terminal moves to the bottom on mobile, sidebar is collapsible
- **Resizable panels** — drag the border to adjust terminal width (desktop) or height (mobile)

## Tech Stack

- React 18
- Vite
- Tailwind CSS v3
- GitHub Pages (via `gh-pages`)

## Getting Started

```bash
npm install
npm run dev
```

## Deployment

```bash
npm run deploy
```

Builds and publishes to the `gh-pages` branch, served at [www.jamesban.com](https://www.jamesban.com).
